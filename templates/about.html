About the Project

The Student Success Predictor is an end-to-end data science application demo.

Project Overview

This application is designed to showcase how Machine Learning can be integrated into a full-stack web environment. It addresses two critical educational challenges:

  • Academic Performance: Predicting how student backgrounds and study habits influence final grades.
  • Retention Management: Identifying students at risk of dropping out based on early behavioral indicators like attendance and initial marks.

Tech Stack Used

Backend
Python Flask
Frontend
HTML5, CSS3, JS
Charts
Chart.js
ML Logic
Simulated Simulation

Objective

The primary goal is to provide a user-friendly interface for educators to interact with predictive models, making data-driven decisions more accessible in the classroom.

# Student Performance & Dropout Prediction System

A full-stack data science web application built with **Python Flask** and modern **HTML/CSS/JS**. This system demonstrates how machine learning models can be integrated into a user-friendly dashboard to predict academic outcomes.

## 🚀 Features

- **Performance Prediction:** Uses study habits and student background to predict final percentages.
- **Dropout Risk Alert:** Identifies at-risk students early.
- **Interactive Dashboard:** Beautiful visualizations using **Chart.js**.
- **Modern UI:** Premium design with glassmorphism, gradients, and smooth animations.

---

## 🛠️ Setup Instructions

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the Application
```bash
python app.py
```

### 3. Access the App
Open **`http://127.0.0.1:5000`**

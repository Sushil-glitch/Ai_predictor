About the Project

The Student Success Predictor is an end-to-end data science application demo.

Project Overview (Static)

This version of the application is optimized for GitHub Pages. The simulation logic has been migrated from Python (Flask) to JavaScript (ES6), allowing it to run entirely in your browser without a backend server.

Architecture Migration

  • Logic: Python Flask API → Client-side JavaScript.
  • Hosting: Cloud Server → GitHub Pages (Static Hosting).
  • Aesthetics: Preserved premium UI with CSS3 and Chart.js.
